Sankey: Custom Sankey Diagram Library
Sankey adalah pustaka Python berbasis Matplotlib yang dirancang untuk mempermudah pembuatan diagram Sankey dengan fitur fokus otomatis pada node tertentu. Alat ini sangat berguna untuk analisis aliran data (seperti penjualan, distribusi logistik, atau alur pengguna) di mana Anda ingin menonjolkan elemen paling dominan dalam data.

🌟 Fitur Utama

- Fokus Otomatis: Secara otomatis menyorot (highlight) node dengan nilai terbesar pada sisi source atau target.
- Kustomisasi Warna: Mendukung palet warna otomatis yang kontras atau daftar warna kustom sesuai kebutuhan brand.
- Fleksibilitas Label: Menampilkan nilai dalam bentuk angka absolut atau persentase secara otomatis.
- Pengurutan Cerdas: Mendukung pengurutan node secara ascending, descending, atau sesuai urutan asli data.
- Visualisasi Bersih: Dilengkapi fitur anti-tabrakan label agar teks tetap terbaca meskipun diagram sangat padat.
